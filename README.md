## 重构计划

该分支『2.0』使用 [gin](https://github.com/gin-gonic/gin) , [gorm](https://github.com/jinzhu/gorm) , [gin-sessions](https://github.com/gin-contrib/sessions) 进行后端重构，前端使用react重构，使用axios和后端 API 进行交互。

- [ ] 架构搭建；
- [ ] 首页；
- [ ] 登录；
- [ ] 获取article；
- [ ] 新建article；
- [ ] 编辑article；
- [ ] 删除article;
- [ ] 获取分类;
- [ ] 新建分类;
- [ ] 编辑分类;
- [ ] 删除分类;
- [ ] 插件机制;
- [ ] 后台管理框架；
